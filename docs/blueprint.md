# **App Name**: Pandey Taxi Service

## Core Features:

- Home Page: Display a landing page with a hero section, service highlights, key features, and a footer. It acts as the entry point to the website.
- Booking Form: A comprehensive form to collect booking details: name, contact, pickup and drop-off points, date and time.
- Contact Form: A fully functional contact form on the Contact page for user inquiries.
- Form Validation: Client-side validation for both booking and contact forms to ensure data accuracy before submission.
- Data Submission: API endpoints to handle the form submission using fetch and POST requests, storing all data in a database like MongoDB or Firebase.
- User Notifications: Display a toast notification, confirming the reception of their inquiry. Uses the AI Tool that decides when to display messages related to submission status.

## Style Guidelines:

- Primary color: Navy blue (#1A237E) to convey trust and professionalism in transportation.
- Background color: Light blue-gray (#E8EAF6) to create a clean and calming interface.
- Accent color: Sky blue (#0277BD) to highlight key interactive elements and calls to action.
- Headline font: 'Space Grotesk' sans-serif for headers.
- Body font: 'Inter' sans-serif for body text.
- Use consistent, modern icons for navigation and key features (e.g., map markers for locations, clocks for timing, etc.).
- A clean, grid-based layout to ensure responsiveness across devices, prioritizing key content areas on each page.
- Subtle transitions and animations on form interactions to provide visual feedback and enhance user experience (e.g., smooth scrolling, fading effects on content reveal).
